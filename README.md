#### 使用警告
在模拟器上使用输入法时，偶发在设置界面选择默认输入法时卡死，原因未知。若遇到到此种情况，可以手动卸载输入法应用后重新安装。

#### 关于词库
- 1.对开源的HanLP-1.x、funNLP等项目的中文语料进行了中文转拼音处理。
- 2.从搜狗细胞词库导入了《原神》、《王者荣耀》、《剑网三》等词库。
- 3.参考1000个最常用汉字使用频率排名优化了词库单词排序。


ps:目前由于从多个来源导入了词库，词库还有些错误的地方待后续修复。
